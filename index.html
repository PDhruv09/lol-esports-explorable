League of Legends Esports - Explorable Explanations

Analyzing trends, strategies, and performance in LoL Esports.

What is being visualized?

This project presents a set of interactive visualizations aimed at analyzing key aspects of League of Legends (LoL) esports matches. The dataset used is sourced from Oracle's Elixir, containing detailed game statistics from professional matches played in 2022.

Why is it being visualized?

The goal of these visualizations is to uncover strategic insights and patterns in LoL esports. By examining aspects such as champion picks, gold differential, game length, objective control, and win rate distribution, we can better understand the factors contributing to victory in competitive play.

How is it being visualized?

Each visualization uses D3.js to create interactive charts, allowing users to explore different dimensions of the dataset. Users can filter and adjust parameters to gain a deeper understanding of each aspect being analyzed. The navigation bar provides access to each visualization for a comprehensive exploration.
